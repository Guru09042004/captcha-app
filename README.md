# 🔐 CaptchaApp
A Spring Boot CAPTCHA generator with audio support.

## Features
- 6-character CAPTCHA (R always included)
- Audio CAPTCHA + WAV export
- REST API for generation & validation

## Run
mvn spring-boot:run
App starts at http://localhost:8080
